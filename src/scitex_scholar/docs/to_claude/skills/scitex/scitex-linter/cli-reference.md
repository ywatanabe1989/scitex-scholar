# CLI Reference

```bash
scitex-linter check [path]              # Check files/directory
scitex-linter check src/ --category io  # Filter by category
scitex-linter list-rules                # List all rules
scitex-linter list-rules --category stats

scitex-linter mcp start                 # Start MCP server
scitex-linter mcp doctor                # Diagnose MCP
scitex-linter skills list               # List skills
```
